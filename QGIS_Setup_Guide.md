# QGIS Setup Guide — SCEIN Fellowship Live Dataset
## Connect scraped_data.csv to QGIS as a Live Map Layer

---

## Prerequisites
- QGIS installed (free at https://qgis.org/en/site/forusers/download.html)
- `scraped_data.csv` generated by running `run_scraper.bat`

---

## Step 1 — Add the CSV as a Layer

1. Open **QGIS**
2. In the top menu, click **Layer → Add Layer → Add Delimited Text Layer**
3. Click the `...` browse button and navigate to your `scraped_data.csv` file
4. QGIS will auto-detect settings. Confirm:
   - **File Format**: CSV (comma separated values) ✅
   - **X field**: `longitude` ✅
   - **Y field**: `latitude` ✅
   - **Geometry CRS**: EPSG:4326 - WGS84 ✅
5. Click **Add**, then **Close**

You will now see your permits, incentives, and regulations plotted as points on the map.

---

## Step 2 — Add a Base Map (So You Can See the Map)

1. In the **Browser Panel** (left side), expand **XYZ Tiles**
2. Double-click **OpenStreetMap**
3. Drag it below your CSV layer so it appears as the background

---

## Step 3 — Style by Type (Permits vs Incentives)

1. Right-click your CSV layer → **Properties → Symbology**
2. At the top dropdown, change **Single Symbol** to **Categorized**
3. Set **Column** to `type`
4. Click **Classify** at the bottom
5. QGIS will auto-assign colors. Customize:
   - **Permit** → Blue
   - **Incentive** → Green
   - **Regulation** → Orange
6. Click **OK**

---

## Step 4 — Set Up Auto-Refresh (Live Update)

So QGIS automatically reloads the CSV when you run the scraper:

1. Right-click your CSV layer → **Properties → General**
2. Scroll down to find **Data source** section
3. Check **Refresh layer on interval** 
4. Set interval to **60 minutes** (or your preference)
5. Click **OK**

Now when you run `run_scraper.bat`, QGIS will reload the data automatically.

---

## Step 5 — Add Labels (Show County Name on Map)

1. Right-click your CSV layer → **Properties → Labels**
2. Select **Single Labels**
3. Set **Label with** to `county`
4. Adjust font size to **8pt**
5. Click **OK**

---

## Step 6 — Filter by Priority (Optional)

To show only HIGH priority items:
1. Right-click layer → **Filter**
2. Enter: `"priority" = 'HIGH'`
3. Click **OK**

To show all: clear the filter.

---

## Step 7 — Save Your QGIS Project

1. **File → Save As**
2. Save as `SCEIN_Dashboard.qgz` in your project folder

Next time you open QGIS, open this file — your layer and styles are saved.

---

## Weekly Workflow (After Setup)

Every Monday, Task Scheduler will:
1. 🤖 Automatically run `scraper.py` at 8:00 AM
2. 📄 Update `scraped_data.csv` with fresh data
3. 🗺️ QGIS auto-refreshes the map layer

**You don't need to do anything manually!**

---

## Troubleshooting

| Problem | Solution |
|---|---|
| Points not showing on map | Check latitude/longitude columns have values |
| CSV layer shows no data | Re-run run_scraper.bat first |
| Points in wrong location | Make sure X=longitude, Y=latitude (not reversed) |
| Task not running | Right-click setup_task_scheduler.bat → Run as Administrator |
| Scraper errors | Check scraper.log file for details |

## Column Reference (for QGIS Attribute Table)

| Column | Description |
|---|---|
| `type` | Permit / Incentive / Regulation |
| `parameter_type` | Solar Permits, Financial Incentive, etc. |
| `dataset_name` | Name from your Excel tracker |
| `county` | County name |
| `state` | State name |
| `latitude` | GPS latitude (for QGIS mapping) |
| `longitude` | GPS longitude (for QGIS mapping) |
| `source_url` | Original link |
| `priority` | HIGH / MEDIUM / LOW |
| `page_title` | Title scraped from the website |
| `page_live` | True/False — is the page still active? |
| `page_status` | HTTP status code (200 = OK) |
| `dates_found` | Any dates found on the scraped page |
| `fees_found` | Any fee information found |
| `contact_found` | Phone number if found |
| `last_scraped` | Timestamp of last scrape |
| `scrape_error` | Error message if scraping failed |
